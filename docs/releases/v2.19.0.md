# Dynrinth v2.19.0

## Resumo

Release base do repositório reorganizado para Fabric moderno e Paper portatil, com core compartilhado embutido nos jars.

## Compatibilidade desta linha

- Fabric: `1.21.11`
- Paper/Bukkit baseline: `1.16.5+`
- Java do core compartilhado: `8`

## Destaques

- Empacotamento correto do `common` nos artefatos finais.
- Validacao de paths para evitar escrita e remocao fora de `mods/` e `plugins/`.
- Tracker registra apenas arquivos realmente baixados.
- Falhas HTTP da API deixam de ser tratadas como falso `not found`.
- Paper convertido para base Bukkit/Spigot portatil.

## Artefatos

- `fabric/build/libs/dynrinth-fabric-2.19.0.jar`
- `paper/build/libs/dynrinth-paper-2.19.0.jar`
